What is Memory Trail?

Memory Trail is a light-weight memory game where your goal is to memorize a path drawn on a grid and then trace it back as accurately as you can. Levels gradually become longer and more complex, challenging your visual memory and focus.

Gameplay

Simple idea, growing challenge

  • First, watch how the trail moves through the grid.
  • When the trail disappears, draw the same path from memory.
  • Each level increases length or complexity of the trail.
Play style

Quick and focused sessions

  • Short levels – great for breaks or commuting.
  • Clear and minimal interface, no unnecessary menus.
  • Learn the rules in seconds, master them over time.
